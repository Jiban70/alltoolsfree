What's the difference between GPA and CGPA?

GPA is for a single semester, while CGPA (Cumulative GPA) is the average of all semesters completed.

Does this work for Nepali/Indian grading systems?

Yes! Convert percentage to 4.0 scale: 90-100% = A (4.0), 80-89% = B (3.0), 70-79% = C (2.0), 60-69% = D (1.0), Below 60% = F (0.0).

What is a good GPA?

3.5+ = Excellent, 3.0-3.49 = Good, 2.5-2.99 = Satisfactory, Below 2.5 = Needs improvement.
